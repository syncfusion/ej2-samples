This demo explains how the Gantt Chart schedules project in UTC timezone.

This sample demonstrates how the Gantt Chart schedules project tasks using the UTC timezone, ensuring consistent date and time handling across global teams.

When a user sets any timezone, dates are converted based on the value set to the timezone property of Gantt control.

Injecting Module:

Gantt component features are segregated into individual feature-wise modules. To use selection feature, inject the Selection module using the Gantt.Inject(Selection) method. To use toolbar feature, inject the Toolbar module using the Gantt.Inject(Toolbar) method.
