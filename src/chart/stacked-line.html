This Stacked Line Chart example visualizes the family expenses data with a stacked line series in the chart to identify who spent more money in each category.

In this example, you can see how to render and configure the stacked line chart. The stacked line chart visualizes data with y-values stacked one over another in a series order. It shows the relationship between individual values to the total sum of points.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use stacking line series, we need to inject StackingLineSeries module using Chart.Inject(StackingLineSeries) method.

More information on the stacked line series can be found in this documentation section.
