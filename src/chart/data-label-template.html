This sample illustrates data label template support in charts. Each data label for each point is rendered using the template.

The TypeScript Charts data label template feature allows you to change the appearance and behavior of the data label for each data point using the template property in ChartDataLabel.


Injecting Module

To use dataLabel, we need to inject dataLabel module using Chart.Inject(dataLabel) method.
