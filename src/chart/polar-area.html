Injecting Module

Chart component features are segregated into individual feature-wise modules. To use area drawtype in Polar series and Radar series, we need to inject PolarSeries, RadarSeries and AreaSeries module using Chart.Inject(PolarSeries, RadarSeries, AreaSeries) method.

More information on the polar and radar chart with an area series can be found in this documentation section.
