This sample demonstrates datalabel template support. In this sample, images are placed as datalabel by means of templates.

Label content can be formatted by using the template option in dataLabel. Inside the template, you can add the placeholder text ${point.y}M to display corresponding data points value.


To use dataLabel, we need to inject dataLabel module using Chart.Inject(dataLabel) method.
