This bar chart example visualizes GDP data by country for the year 2017 with a default bar series.

In this example, you can see how to render and configure the bar chart. The bar chart is similar to the column chart, but the orientation of the y-axis is horizontal rather than vertical.

Tooltips are enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use bar series, we need to inject BarSeries module using Chart.Inject(BarSeries) method.

More information on the bar series can be found in this documentation section.
