This Range Area Chart example visualizes minimum and maximum temperatures of different days with default range area series.

In this example, you can see how to render and configure the range area chart. This chart is used to display continuous data points as a set of lines varying between high and low values over time intervals and across different categories.


Injecting Module

chart component features are segregated into individual feature-wise modules. To use rangearea series, we need to inject RangeAreaSeries module using chart.Inject(RangeAreaSeries) method.

More information on the range area series can be found in this documentation section.
