Series Type:

This sample demonstrates polar series with column type mobile market subscriptions in differnt countries. Series type can be changed by Series Type

In this example, you can see how to render and configure the column type charts. Column type charts are used for comparing the frequency, count, total or average of data in different categories. You can use border, fill properties to customize the vertical rect. dataLabel is used to represent individual data and its value.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.

Injecting Module

Chart component features are segregated into individual feature-wise modules. To use column drawtype in Polar series and Radar series, we need to inject PolarSeries and RadarSeries module using Chart.Inject(PolarSeries, RadarSeries) method.

More information on the column series can be found in this documentation section.
