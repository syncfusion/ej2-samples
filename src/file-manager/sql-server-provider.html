This sample demonstrates how to utilize the SQL server file system provider with File Manager control. To run the service, configure the SQL server database connection using the SetSQLConnection method to set the connection name, table name and rootId of the SQL table.

The File Manager control is used to explore a file system through a web application, similar to the Windows Explorer for Windows. It supports all the basic file operations such as create, rename, delete, and more.

Note: File Manager’s upload functionality is restricted in online demos for security reasons. To work with upload functionality, please download SQL Database Provider from the GitHub repository.
