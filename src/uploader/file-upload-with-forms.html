This example demonstrates the Uploader control supported with HTML form upload. Fill the mandatory details in a form and click the submit button.

The Uploader control works in synchronous mode using HTML form. When the end-user submits the form, the selected files are submitted to server with the name attribute of input element.

More information on the form support can be found in this documentation section.
