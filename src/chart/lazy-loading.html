This sample illustrates lazy laoding feature in chart. Loads data for chart on demand.

In this example, you can see how to load data for chart on demand. Chart will fire the scrollEnd event, in that can udpate the chart with required data based on point length and axis range.

ScrollBar is enabled in the sample and ScrollBar module injected to the chart.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use lazy laoding need to inject ScrollBar and Zoom module. Chart.Inject(ScrollBar, Zoom) method.
